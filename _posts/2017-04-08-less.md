## 1. 变量

```less
@color: red;
@img: "/img";
div {
 background: @color url("@{img}/bg.jpg");
}
```
其中 `@{img}` 为插入字符串，需使用花括号括起来
 
输出:

```css
div {
  background: red url("/img/bg.jpg")
}
```

甚至可以用变量名定义为变量:

```less
@fnord: "I am fnord.";
@var: 'fnord';
content: @@var;
```

`@@var` 等价于` @(@var)` 等价于 `@(fnord)` 等价于 `@fnord` 等价于 `"I am fnord."`
